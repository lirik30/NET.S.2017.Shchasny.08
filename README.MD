# Customer class
Класс Customer предоставляет возможность форматирования своих экземпляров. По умолчанию(файл app.config!) установлена en-US культура. Существует возможность создать свой собственный провайдер формата для этого класса. Классом предусмотрены следующие возможности форматирования:
 - G: "Имя, Доход, Номер телефона";
 - N: "Имя";
 - P: "Номер телефона";
 - R: "Доход";
 - NP: "Имя, номер телефона";
 - NR: "Имя, доход";
 - PR: "Номер телефона, доход";
